Write a Swift app that wraps a given website domain (use “chat.openai.com”). Any interactions within that website domain should open in the app’s WKWebView; any URLs loaded that aren’t in that domain will open the URL in Safari. The app should have a single view, a WKWebView, which delegates to the AppDelegate. The UIWindow and view should be set up completely via code, no xib or storyboard files.

Here is an example of how you could wrap the website "chat.openai.com" in a Swift app using a WKWebView:

import UIKit
import WebKit

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate, WKNavigationDelegate {

    var window: UIWindow?
    var webView: WKWebView!

    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        // Set up the window and web view
        window = UIWindow(frame: UIScreen.main.bounds)
        webView = WKWebView(frame: window!.frame)
        webView.navigationDelegate = self
        window!.addSubview(webView)

        // Load the website
        let url = URL(string: "https://chat.openai.com")!
        webView.load(URLRequest(url: url))

        window!.makeKeyAndVisible()
        return true
    }

    func webView(_ webView: WKWebView, decidePolicyFor navigationAction: WKNavigationAction, decisionHandler: @escaping (WKNavigationActionPolicy) -> Void) {
        // If the URL is within the chat.openai.com domain, allow it to be loaded in the app's web view
        if navigationAction.request.url?.host == "chat.openai.com" {
            decisionHandler(.allow)
        } else {
            // If the URL is not within the chat.openai.com domain, open it in Safari
            UIApplication.shared.open(navigationAction.request.url!)
            decisionHandler(.cancel)
        }
    }
}

This app has a single view, a WKWebView, which is set up completely in code and delegates to the AppDelegate. The app loads the website "chat.openai.com" in the WKWebView and uses the WKNavigationDelegate method webView(_:decidePolicyFor:decisionHandler:) to determine whether to allow a URL to be loaded in the app's web view or to open it in Safari. If the URL is within the "chat.openai.com" domain, it is allowed to be loaded in the app's web view. If the URL is not within that domain, it is opened in Safari.
